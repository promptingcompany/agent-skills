# Installation Guide

## Claude Code (CLI)

```bash
cp -r skills/tpc-existing-product-ui ~/.claude/skills/
```

Claude Code discovers the skill automatically. Use it alongside the target
repository's design guidance and browser-verification workflow.

## claude.ai (Project Knowledge)

1. Open the project and go to **Project Knowledge > Add content**.
2. Add `SKILL.md` and `workflows/implement-existing-product-ui.md`.

## Uninstalling

```bash
rm -rf ~/.claude/skills/tpc-existing-product-ui
```
