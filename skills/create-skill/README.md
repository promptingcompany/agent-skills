# Create Skill

Create Skill helps an agent add a new skill package to this repository. It requires the agent to read the root `SPEC.md` first, then scaffold the files that the repository expects under `skills/<skill-name>/`.

## When to use

Use this skill when you want to:

- Add a new reusable skill to `agent-skills`
- Turn a workflow, prompt, or operating procedure into a packaged skill
- Generate a `SKILL.md`, `README.md`, `INSTALL.md`, `metadata.json`, and workflow file set
- Make sure a new skill follows the repository's structural requirements

## Included workflows

- [Create New Skill](workflows/create-new-skill.md): read `SPEC.md`, define the skill contract, scaffold package files, update the root README, and validate the result.
