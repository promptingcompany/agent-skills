# Analyze Experiment

Turn a completed experiment iteration into a portable data dump, an honest, evidence-backed markdown report, and a branded client-ready PDF — the third skill in the experiment suite: **setup → run → analyze**.

## Workflows

| Workflow | Triggers |
|---|---|
| Analyze Experiment (always produces the data dump; delivers the report as `md`/`pdf`/`html` per your choice, plus an optional chat-only setup analysis — both asked up front) | "analyze my experiment", "write the report", "experiment report", "analyze the results", "report gen", "generate a pdf", "client-ready report", "like the previous pilot PDF" |
| Deep Evidence Report | "evidence-backed report", "no-fluff report", "deep report", "prioritized recommendations" |
